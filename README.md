# TweetSC
Tweet Spell Checker (TweetSC) is a spell checker for tweets. It corrects errors in a Spanish short text, e.g., a Tweet.

## How to compile / Requirements

- Java 1.8
- maven? gradle?

```
git clone https://github.com/jmorenov/TweetSC
mvn clean install 
```
